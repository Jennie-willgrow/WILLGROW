👩‍🏫 Teacher Management
Manage teacher accounts and roles
1
Supabase 로그인 계정 생성
Supabase Dashboard → Authentication → Users → Add User
이메일: [이름소문자]@willgrow.local / 비밀번호 설정 후 선생님께 전달
2
아래 목록에서 선생님 추가
위 + Add Teacher 버튼 → 이름과 역할 입력 → Save
이름은 로그인 이메일 앞부분과 반드시 일치해야 함
3
로그인 정보 전달 & 확인
웹앱 주소와 이메일/비밀번호를 선생님께 공유
선생님이 로그인 후 메뉴가 올바르게 보이는지 확인
1
아래 목록에서 선생님 삭제
해당 선생님 → Edit → Delete
삭제 즉시 역할 기반 메뉴 접근 차단됨
2
Supabase 로그인 계정 삭제
Supabase Dashboard → Authentication → Users
해당 선생님 계정 찾아 Delete User
